CNN 모델
합성곱 레이어
필터, 수용영역, 풀링, 스트라이드 합성곱,

전이학습이란 이미 학습된 모델의 가중치를 새로운 문제에 활용하는 방법.

[cnn의 특직 추출 구조]
입력 이미지 >> 특징 추출( 합성곱 + 풀링) >> 분류(Fully connected layers) >> 출력

왜 전이학습이 효과적?
-> cnn이 학습하는 저수준 특징들은 매우 범용적이어서 다양한 이미지셋에서 공통으로 유용하다.
-> 반면, 마지막 분류층은 데스크에 특화된 정보(예를 들어 imagenet의 100개 클래스)를 담고 있어 새로운 데스크에 맞게 재학습이 필요하다.
Linear probing은 분류층만 학습. 시간 빠름. 데이터가 적을 때 유용. 기본 성능 확보. 과적합 가능성 낮음. 학습 파라미터는 분류층만(전체의~1%)
Fine tuning은 일부 또는 전체 모델을 추가 학습. 학습 속도 느림. 데이터가 방대하거나 컴퓨팅 자원이 있을 때 유용. 과적합 가능성 높음(데이터가 적을때). 최고 성능 달성. 학습 파라미터는 전체모델(100%)
