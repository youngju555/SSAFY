기계학습 라이브러리(scikit-learn 등)에 의존하지 않고, **오직 NumPy 행렬 연산만을 사용하여 선형 회귀 모델의 원리를 밑바닥부터 직접 구현(From Scratch)하고 비교하는 과정**을 담고 있습니다.

---

## 1. 데이터 전처리 (Data Preprocessing)
모델이 올바르게 학습할 수 있도록 데이터를 정제하고 단위를 맞추는 단계입니다.

* **변수 분리**: 계산이 불가능한 문자/범주형 데이터를 배제하고, 연속형(숫자) 데이터만 추출합니다.
* **표준화(Standardization)**: 각 변수의 단위가 모델에 미치는 영향을 제거하기 위해, 평균을 0, 표준편차를 1로 맞춥니다. 수식은 $z = \frac{x - \mu}{\sigma}$ 입니다.

```python
# 연속형 변수 추출 및 NumPy 배열 변환
continuous_cols = df.select_dtypes(include=['float64','int64']).columns.tolist()
X_raw = df[continuous_cols].values

# 표준화 연산 (0으로 나누는 오류 방지 포함)
mu = np.mean(X_raw, axis=0)
std = np.std(X_raw, axis=0)
std = np.where(std == 0, 1, std)
X_norm = (X_raw - mu) / std

# 절편(Bias) 계산을 위한 더미 변수(1) 추가
m = X_norm.shape[0]
X_b = np.c_[np.ones((m, 1)), X_norm]
```

## 2. 선형대수를 이용한 해법 (Analytical Solution)
수학적 공식을 사용하여 오차가 가장 적은 최적의 가중치($\theta$)를 단번에 계산하는 방식입니다.

* **정규 방정식 (Normal Equation)**: 수학적으로 완벽한 정답을 내지만, 특성이 많아지면 역행렬 연산에 과부하가 걸립니다.
* **특이값 분해 (SVD)**: 역행렬이 존재하지 않는 꼬인 데이터에서도 안정적으로 근사치를 구해내는 실무적인 수학 기법입니다. (`np.linalg.lstsq`가 내부적으로 사용하는 방식)

```python
# 방법 A: 정규 방정식
theta_normal = np.linalg.inv(X_b.T @ X_b) @ X_b.T @ y

# 방법 B: 최소제곱법 (SVD 기반)
theta_lstsq, residuals, rank, s = np.linalg.lstsq(X_b, y, rcond=None)
```


## 3. 경사 하강법을 이용한 해법 (Numerical Solution)
데이터가 너무 커서 행렬을 한 번에 계산할 수 없을 때, 정답을 향해 '조금씩 반복해서' 이동하는 최적화 방식입니다. 딥러닝의 핵심 원리이기도 합니다.

* **기본 경사 하강법 (Batch GD)**: 전체 데이터를 보고 한 번 이동합니다.
* **순차적 미니배치 (Mini-batch GD)**: 데이터를 작게 쪼개서(배치) 잦은 빈도로 업데이트하며, 메모리 효율이 좋습니다.

```python
# 경사 하강법 가중치 업데이트 핵심 로직
y_pred = X_batch @ theta
error = y_pred - y_batch

# 그래디언트(기울기) 계산
gradient = (2 / len(X_batch)) * X_batch.T @ error

# 학습률(alpha)만큼 반대 방향으로 이동하여 가중치 갱신
theta -= alpha * gradient
```


## 4. 하이퍼파라미터 실험 (Learning Rate Analysis)
경사 하강법에서 이동하는 보폭인 **학습률($\alpha$)**이 모델 성능에 미치는 영향을 분석했습니다.

* **결과 검증**:
    * 학습률이 너무 작으면($10^{-5}$) 학습이 정체됩니다.
    * 적절하면($0.05$) 가장 빠르게 최적점에 수렴(Convergence)합니다.
    * 너무 크면($1.0$) 숫자가 무한대로 커지며 에러(NaN)를 뱉고 발산(Divergence)합니다.

---

결론적으로 **"수학 공식(선형대수)으로 한 번에 푸는 것이 정확하지만, 대용량 데이터 환경을 대비해 점진적으로 정답을 찾아가는 경사 하강법의 원리도 알아야 한다"**는 것을 실습을 통해 증명하고 있습니다.
